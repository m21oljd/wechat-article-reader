🤖 AI深度分析

📌 使用说明

  1. 复制微信公众号文章的链接（在文章页面点击右上角「...」→「复制链接」）
  2. 将链接粘贴到上方的输入框中
  3. 点击「解析文章」按钮获取文章内容
  4. 选择已有模板或输入自定义分析提示词
  5. 点击「AI分析」按钮，等待AI生成分析报告
  6. 如果自定义提示词好用，可以点击「另存为新模板」保存起来
提示：使用AI分析功能需要配置 DeepSeek API 密钥。请在项目目录下创建 .env 文件，添加：
DEEPSEEK_API_KEY=your_api_key_here
